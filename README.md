# nexus-tv-updates
Update files for Nexus TV IPTV PLayer
NEXUS TV - INSTALLATION GUIDE

Why does Windows block this app?
- Nexus TV is not code-signed (requires $200/year certificate)
- Windows Defender flags ALL unsigned software as "potentially unsafe"
- This is a false positive - the app is safe

How to install:

WINDOWS DEFENDER:
1. Download Nexus TV Setup.exe
2. Windows will show "Windows protected your PC"
3. Click "More info"
4. Click "Run anyway"
5. Follow installer
